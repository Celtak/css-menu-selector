# css-menu-selector
A menu selector using a css class and vanilla javascript

## Usage

### Example

#### html

```html
<nav>
  <div id="nav-1">Button-1</div>
  <div id="nav-2">Button-2</div>
  <div id="nav-3">Button-3</div>
  <div id="nav-4">Button-4</div>
</nav>
```

#### css

```css
nav {
  display:flex;
}
```
